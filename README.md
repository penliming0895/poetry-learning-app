# 初三语文古诗词背默

一款帮助初三学生背诵和默写古诗词的 Web 应用，支持 PWA 离线使用，可添加到桌面。

## ✨ 功能特点

### 📖 背诵练习模式
- 逐句背诵，智能提示
- 语音朗读，多种音色可选
- 语速调节，适应不同需求
- 实时反馈，轻松掌握

### ✍️ 完整测试模式
- 全篇默写，智能评分
- 逐句对照，精准反馈
- 语音朗读，辅助记忆
- 错题记录，重点复习

### 📅 每日背默
- 自动轮换，每天5首
- 进度追踪，一目了然
- 坚持打卡，养成习惯

### 🏆 成就系统
- 16种成就，激励学习
- 多种稀有度，增加趣味
- 自动解锁，即时反馈

### ❌ 错题本
- 自动记录错误
- 分类整理，高效复习
- 掌握标记，灵活管理

### 🎯 PWA 支持
- 可添加到桌面
- 支持离线使用
- 像原生 App 一样

## 🚀 快速开始

### 在线使用
访问部署链接即可使用，无需安装。

### 安装到桌面
- **iOS**: Safari → 分享 → 添加到主屏幕
- **Android**: Chrome → 菜单 → 添加到主屏幕
- **电脑**: Chrome/Edge → 地址栏安装图标

详细部署指南请查看 [DEPLOY.md](./DEPLOY.md)

## 🛠️ 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: React 19 + Tailwind CSS 4
- **组件**: shadcn/ui
- **PWA**: next-pwa
- **语音**: coze-coding-dev-sdk
- **包管理**: pnpm

## 📚 数据说明

- **诗词数量**: 60首初三必背古诗词
- **每日任务**: 每天自动轮换5首
- **存储方式**: localStorage（本地存储，保护隐私）

## 🔒 隐私说明

- 所有学生数据存储在本地浏览器中
- 不上传任何个人数据到服务器
- 清除浏览器数据会丢失进度

## 📖 使用说明

### 初次使用
1. 选择学习模式（推荐先从"每日背默"开始）
2. 点击"开始"按钮
3. 按照提示完成背诵/默写
4. 查看结果和成就

### 练习建议
- 每天完成5首每日背默
- 定期复习错题本
- 收集成就增加趣味
- 坚持打卡养成习惯

## 🆘 常见问题

### Q: 语音朗读不工作？
A: 语音功能需要网络连接，确保设备已联网。

### Q: 进度丢失了怎么办？
A: 检查浏览器是否清除了 localStorage 数据，或点击页面右上角的"清除缓存"按钮。

### Q: 如何添加更多诗词？
A: 编辑 `src/data/poetryData.ts` 文件，添加新的诗词数据。

### Q: 能否导出学习数据？
A: 当前版本不支持，后续版本会添加数据导出功能。
